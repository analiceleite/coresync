
# CoreSync

**CoreSync** é uma ferramenta interna desenvolvida para centralizar e facilitar a gestão de reuniões, perfis de colaboradores e treinamentos obrigatórios dentro de uma empresa. Com uma interface moderna e funcionalidades que otimizam o fluxo de trabalho, o CoreSync ajuda colaboradores e gestores a manterem-se organizados e em conformidade com os processos críticos do ambiente corporativo.

## Funcionalidades Principais

### 1. Reuniões

* Agendamento de bate-papos e reuniões.
* Geração automática de atas de reuniões.

### 2. Perfil do Colaborador 

* Visualização do perfil completo de cada colaborador.
* Edição dos dados do perfil.

### 3. Treinamentos Obrigatórios

* Trilha de aprendizagem obrigatória para cada colaborador.

## Tecnologias Utilizadas

### Backend

* **[Django Rest Framework](https://www.django-rest-framework.org/)** : Estrutura poderosa e flexível para a criação de APIs, usada para gerenciar as operações do backend.
* **Python** : Linguagem de programação utilizada para a construção do backend.

### Frontend

* **[React](https://reactjs.org/)** : Biblioteca JavaScript para a construção de interfaces de usuário interativas e dinâmicas.
* **[Styled Components](https://styled-components.com/)** : Utilizado para a estilização do projeto com CSS-in-JS, garantindo flexibilidade e manutenção do design.

## Estrutura do Projeto

* **Backend** : O backend foi desenvolvido utilizando Django Rest Framework, oferecendo uma API robusta para gerenciar todas as funcionalidades do sistema, como criação de perfis, agendamento de reuniões e gestão de treinamentos.
* **Frontend** : O frontend foi construído com React e Styled Components, garantindo uma interface fluida e responsiva. Os componentes visuais são criados dinamicamente com base nos dados do backend.
